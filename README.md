# Система ведения заказов (Вариант №5)

Проект для учёта оптовых продаж товаров.

## 📋 Описание
Информационная система для отслеживания финансовой стороны работы оптовой компании.

## 🗄️ Сущности базы данных
| Сущность | Атрибуты |
|----------|----------|
| **Products** | цена, справочная информация, признак доставки |
| **Customers** | наименование, адрес, телефон, контактное лицо |
| **Orders** | заказчик, количество товара, дата покупки |

## 🚀 Как запустить
1. Клонируйте репозиторий
2. Откройте решение в Visual Studio
3. Запустите проект
